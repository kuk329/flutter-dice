## What you will learn

#### - 상태가 변하는 부분은 stateful, 변하지 않는 부분은 stateless를 사용해서 만든다. 한 화면에 구역별로 필요한걸 적절히 사용하면 된다. 
#### - math library를 사용해서 random으로 숫자를 얻는다. -> import 'dart:math'
#### - ex) 1~6까지 숫자 랜덤하게 생성 -> diceNumber = Random().nextInt(6)+1
